# Portafolio v3 (clean / minimal)

## Archivos
- `index.html` (estructura)
- `styles.css` (estilos)
- `assets/` (CV, plan de pruebas, foto)

## Cómo poner tu foto
Opción A (recomendada):
1) Subí una imagen como `assets/profile.jpg` (o `.png`)
2) En `index.html`, buscá esta línea:
   `src="assets/profile-placeholder.png"`
   y cambiála por:
   `src="assets/profile.jpg"`

Opción B:
Reemplazá el archivo `assets/profile-placeholder.png` por tu imagen (mismo nombre).

## Publicar en GitHub Pages
1) Subí TODO el folder (incluye `assets/`).
2) Settings → Pages → Deploy from branch → `main` / root.
